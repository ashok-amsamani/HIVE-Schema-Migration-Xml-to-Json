# Schema-Migration-Xml-to-Json

Listed steps about how to migrate xml data to json data.

Follow steps given in script.txt. It has HIVE/Linux and Hadoop scripts. You should have a knowledge on where to run what, to try this excercise.

It covers:

  1. Getting xml and Json serde from Maven (wget).
  2. Adding serde jars in hive session.
  3. Creating xml and json serde tables.
  4. Moving data from linux to hive table location.
